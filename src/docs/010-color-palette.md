# Color Palette: Landing Page V2

This document defines the official color palette for the "Firecrawl" aesthetic redesign of the StartupAI Landing Page V2. The design system relies on a stark, high-contrast foundation of black and white, punctuated by a vibrant "Brand Orange" for primary actions and highlights.

## Primary Brand Colors

Used for buttons, links, active states, and key visual accents.

| Color Name | Hex Code | Tailwind Class | Usage |
|:--- |:--- |:--- |:--- |
| **Brand Orange** | `#FF6A3D` | `text-[#FF6A3D]`, `bg-[#FF6A3D]` | Primary buttons, active icons, text highlights, gradient stops. |
| **Orange Hover** | `#E55A2D` | `hover:bg-[#E55A2D]` | Hover state for primary buttons. |
| **Orange Surface** | `#FFF7ED` | `bg-[#FFF7ED]` | Subtle backgrounds for badges (Orange-50 equivalent). |
| **Orange Border** | `#FFEDD5` | `border-[#FFEDD5]` | Borders for orange-tinted elements (Orange-100 equivalent). |

## Neutral Scale

Used for text, backgrounds, borders, and structural elements.

| Color Name | Hex Code | Tailwind Class | Usage |
|:--- |:--- |:--- |:--- |
| **Obsidian** | `#111827` | `text-[#111827]`, `bg-[#111827]` | Main headings, bold text, solid black buttons, code headers. |
| **Slate** | `#6B7280` | `text-[#6B7280]` | Body text, secondary descriptions, inactive labels. |
| **Ash** | `#9CA3AF` | `text-[#9CA3AF]` | Footer text, placeholders, tertiary information. |
| **Border Gray** | `#E5E7EB` | `border-[#E5E7EB]` | Card borders, dividers. |
| **Surface Gray** | `#F9FAFB` | `bg-[#F9FAFB]` | Main page background (Gray-50). |
| **Canvas White** | `#FFFFFF` | `bg-white` | Card surfaces, navbar, input fields. |

## Gradient Definitions

**Top Banner Gradient:**
`bg-gradient-to-r from-[#FF6A3D] via-[#FF4D4D] to-[#FF6A3D]`
*(Note: This banner was removed in the final V2, but the gradient remains available for promotional accents).*

**Hero Glow:**
Radial gradient from `#FFFFFF` to transparent, used to lift text off the technical grid.
