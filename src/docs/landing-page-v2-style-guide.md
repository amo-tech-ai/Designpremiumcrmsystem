# StartupAI Landing Page V2 Style Guide

This document outlines the design system, color palette, and component styles used in the `LandingPageV2` and associated components. The design follows a "Firecrawl" aesthetic: clean, technical, and investor-grade, featuring a stark white/gray background with high-contrast Orange accents.

## 1. Color Palette

### Primary Colors
Used for primary actions, brand highlights, and active states.

| Name | Hex Code | Usage |
|------|----------|-------|
| **Brand Orange** | `#FF6A3D` | Primary buttons, active icons, text highlights, gradients |
| **Orange Hover** | `#E55A2D` | Hover state for primary buttons |
| **Orange Fade** | `#FFF7ED` | Backgrounds for badges, icon containers (Orange-50 equivalent) |

### Neutrals
Used for text, borders, and backgrounds.

| Name | Hex Code | Usage |
|------|----------|-------|

## 2. Typography

The design relies on the default Tailwind sans-serif font stack (Inter/System UI) for cleanliness and readability, and a Monospace font for technical data.

- **Headings:** `font-sans font-bold tracking-tight`. 
  - Hero H1: `text-5xl md:text-7xl`
  - Section H2: `text-3xl md:text-4xl`
  - Card H3: `text-xl`
- **Body:** `font-sans text-base leading-relaxed`.
- **Code:** `font-mono text-sm` (for JSON outputs).

## 3. UI Components

### Buttons
**Primary Button:**
- Background: Brand Orange (`#FF6A3D`)
- Text: White
- Radius: `rounded-lg` or `rounded-xl`
- Shadow: `shadow-md`
- Hover: Darker Orange (`#E55A2D`) + `hover:shadow-lg`

**Secondary/Ghost Button:**
- Background: Transparent or White
- Text: Brand Orange (`#FF6A3D`) or Dark Gray (`#111827`)
- Hover: Tinted background (`bg-[#FF6A3D]/5`)

### Cards
Cards (Features, Pricing, Workflow Steps) follow a unified "tech-card" style:
- **Shape:** `rounded-2xl` or `rounded-3xl`
- **Border:** 1px solid `#E5E7EB`
- **Shadow:** `shadow-sm`
- **Background:** White
- **Hover Effect:** 
  - Slight lift (`translate-y` or `scale`)
  - Increased shadow (`shadow-md`)
  - Border color shift (optional, usually to Orange or darker gray)

### Badges / Tags
- **Style:** Pill-shaped `rounded-full`
- **Context:** Used for "Powered by Gemini", "2 Months Free", etc.
- **Visuals:** Light background (`bg-white` or `bg-orange-50`) with a border.

## 4. Visual Effects & Motifs

### Technical Grids (Hero Background)
A subtle backdrop to imply structure and engineering.
```css
/* Fine Grid */
background-image: linear-gradient(to right, #80808008 1px, transparent 1px),
                  linear-gradient(to bottom, #80808008 1px, transparent 1px);
background-size: 24px 24px;

/* Dot Grid Overlay */
background-image: radial-gradient(#CBD5E1 1px, transparent 1px);
background-size: 40px 40px;
opacity: 0.3;
```

### Glows
Soft, blurred gradients used behind the Hero element or Code blocks to add depth without clutter.
- **Color:** White (on gray backgrounds) or Orange/Red gradients.
- **Blur:** `blur-3xl` or `blur-2xl`.

### Glassmorphism
Used on the Sticky Navbar.
- **Class:** `bg-white/90 backdrop-blur-md`

### Motion
Entrance animations using `motion/react`.
- **Fade Up:** `initial={{ opacity: 0, y: 20 }} animate={{ opacity: 1, y: 0 }}`
- **Scale In:** `initial={{ opacity: 0, scale: 0.95 }} animate={{ opacity: 1, scale: 1 }}`

## 5. Iconography
- **Library:** `lucide-react`
- **Style:** Stroke width 2px (default).
- **Coloring:** 
  - Neutral icons: `#6B7280` or `#111827`
  - Active/Highlighted icons: `#FF6A3D`

## 6. Layout Guidelines
- **Max Width:** `max-w-[1200px]` (Standard) or `max-w-[1440px]` (Wide/Workflow).
- **Padding:** `px-6` (Horizontal padding for mobile safety).
- **Section Spacing:** `py-20` or `py-24` (Generous vertical whitespace).
